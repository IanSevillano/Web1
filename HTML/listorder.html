The start attribute

By default, an ordered list will start counting from 1. Use the start attribute to start counting from a specified number:

  1. Coffee
  2. Tea
  3. Milk
  1. Coffee
  2. Tea
  3. Milk

A Nested List

Lists can be nested (list inside list):

  1. Coffee
  2. Tea
    1. Black tea
    2. Green tea
  3. Milk
